# Ticketmaster Lite
## Description
As a user you can:
1. Login as an existing user or create a new account
2. Delete your own account
3. Search for events by a particular city
4. Search for events by a particular artist or sports team
5. Add or remove events to your personal event calendar
## Installation
1. Clone application from my [github repo](https://github.com/cisneros-a/module-one-final-project-guidelines-austin-web-102819).
2. Once in the correct directory from the terminal, install the proper gems using: <br>
`bundle install` 
3. Run the CLI application using: <br>
`ruby bin/run.rb`
